## Operators
* `&&` - and 
* `||` - or
  * `&&` is evaluated before `||`
* `===` - items are exactly the same
  * `==` - **DO NOT USE** items are similar? 3 == '3' would be true 
* `!==` - not equal
* `>`, `<` - greater than, less than
* `>=` - greater than or equal to
